overView Graph with Suspect Flags Enabled


Summary: I pulled the the data for the graph from the remote server at St. Louis using their FRED API.Once I pulled in the data, I converted it into JSON and then stored the sections I needed and discarded the rest. Finally, the data was uploaded into a Google Line Graph. The graph is able to toggle the data by selecting which sections are to be shown based on a set of flags. For a more technical view please view the comments in the index.html file and FRED.js file.
